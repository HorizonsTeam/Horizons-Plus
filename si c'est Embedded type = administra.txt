si c'est Embedded type = administrative_region
	on prend s.administrative_region.name puis on rajouteras en faisant un lien avec le code insee la région en gris
si c'est Embedded type = stop_area
	on prend s.stop_area.name puis après on rajoute en gris pas la région mais l'administrative region auquelle elle est associé : s.stop_area.administrative_regions.name

sinon si c un ville random faut mettre la région, si c quartier banlieue faut mettre la ville associé.
